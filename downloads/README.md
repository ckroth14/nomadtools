# /downloads

Drop the real `nomad-ip-toolbox.zip` (or renamed equivalent) here.

`toolbox.html` links directly to `downloads/nomad-ip-toolbox.zip` with a
`download` attribute — the button already works, it just needs the real file
to land in this folder. If you rename the file or host it elsewhere (GitHub
release, R2, etc.), update the `href` in `toolbox.html`'s download block to
match.

See SETUP.md for the full checklist.
