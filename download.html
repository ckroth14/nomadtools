nomad.ip ToolBox

Windows utility for diagnosing and configuring IP cameras and network devices in the field.

Download for Windows
Windows 10 / 11 · No install required · ~2MB

Before you run it

Windows may show a SmartScreen warning for unsigned apps — this is expected for a small independent tool. Click "More info" then "Run anyway" to continue.

The first time you run it may take longer to start. It will then ask to run as an administrator. This will keep Windows from asking permission every time you change your IP.
